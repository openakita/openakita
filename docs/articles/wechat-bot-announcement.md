## 微信接入有多简单？3 个步骤搞定

### 前置条件

在微信中搜索并开启 **ClawBot 插件**。这是微信官方提供的机器人接口能力，开启后你才能通过微信和OpenAkita聊天。

### 步骤一：在 OpenAkita 中选择微信平台

打开 OpenAkita 设置中心，进入 **IM 通道** 页面，点击 **「引导创建」** 按钮。

在弹出的引导向导中，选择 **微信** 平台：

![选择微信平台](../assets/step1-select-wechat.png)

目前支持的全部 IM 平台一览：微信、飞书、钉钉、企业微信、QQ、Telegram、OneBot。每个平台都有独立的接入向导。

### 步骤二：绑定 Agent

选择你要为这个微信 Bot 绑定的 Agent。默认使用「小秋」（默认 Agent），如果你开启了多 Agent 模式，还可以选择专属的 Agent Profile。

![绑定 Agent](../assets/step2-bind-agent.png)

### 步骤三：设置 Bot ID 和名称

系统会自动生成 Bot ID 和名称，你也可以自定义。Bot ID 创建后不可更改，请确认后继续。

![设置 ID 和名称](../assets/step3-id-name.png)

### 步骤四：扫码登录

这是最关键的一步——系统会生成一个二维码，**用你的微信扫码登录**即可完成绑定。

不需要填写 AppID、不需要配置回调地址、不需要申请公网 IP。

**扫码，完事。**

### 步骤五：开始对话

创建完成后，你会在 IM Bot 配置列表中看到新创建的微信 Bot。点击「启用」，你的 AI 助手就正式上线了。

![IM 通道主页](../assets/step0-im-overview.png)

现在，任何人给你发微信消息，AI Agent 都能自动响应、调用工具、完成任务。

## 引导创建 vs 快速创建

OpenAkita 提供了两种创建 Bot 的方式：

**引导创建（推荐）**：一步步引导你完成平台选择 → Agent 绑定 → 凭据配置 → 平台设置，特别适合第一次接入的用户。

**快速创建**：直接填表创建，适合熟悉各平台 API 配置的开发者。

无论哪种方式，创建完成后都可以随时编辑、暂停、删除。

## 技术细节：微信官方协议长什么样？

对于技术同学，简单聊聊底层实现。

微信 ClawBot 基于腾讯提供的 **iLink Bot API** 协议，核心特性：

- **消息接收**：HTTP 长轮询（`getUpdates`），类似 Telegram Bot API 的设计
- **消息发送**：HTTP API（`sendMessage`），支持文本、图片、文件
- **媒体处理**：CDN 上传/下载 + AES-128-ECB 加解密
- **登录方式**：扫码获取 Bearer Token，无需 OAuth 复杂流程
- **Typing 指示器**：支持 `sendTyping`，让对方看到「正在输入...」

```
API Base: https://ilinkai.weixin.qq.com
CDN Base: https://novac2c.cdn.weixin.qq.com/c2c
```

OpenAkita 在此基础上做了完整的适配：消息收发、图片处理、断线重连、失败重试，开箱即用。
