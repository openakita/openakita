## 三、环境变量配置

**文件路径**: `mvp/api-integration/.env.credentials`

```bash
# 企业微信
WECHAT_CORP_ID=
WECHAT_AGENT_ID=
WECHAT_SECRET=

# 钉钉
DINGTALK_APP_KEY=
DINGTALK_APP_SECRET=

# 飞书
FEISHU_APP_ID=
FEISHU_APP_SECRET=

# 阿里云
ALIYUN_ACCESS_KEY_ID=
ALIYUN_ACCESS_KEY_SECRET=

# Google Calendar
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REFRESH_TOKEN=
```

**安全要求**：
- ⚠️ 敏感信息严禁明文传输
- ⚠️ 使用 AES-256 加密存储
- ⚠️ 参考现有 `credentials.json` 方案

## 四、申请流程

1. **准备材料**：营业执照、身份证、企业邮箱/手机
2. **注册/登录平台**：使用企业账号登录
3. **创建应用/账号**：按平台指引填写信息
4. **配置权限**：勾选所需 API 权限
5. **获取凭据**：记录 AppKey/Secret/Token 等
6. **测试连通性**：@dev-a 进行 API 验证
7. **记录归档**：更新本文档 + 配置文件

## 七、风险与注意事项

1. **实名认证**：阿里云需要企业实名认证，可能需要 1-2 天审核
2. **权限审批**：部分平台应用创建需要企业管理员审批
3. **验证码接收**：确保企业邮箱/手机可正常接收验证码
4. **凭据安全**：Secret/Token 等敏感信息严禁明文传输
