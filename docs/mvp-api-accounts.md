```bash
# ==================== 企业微信 ====================
WECHAT_CORP_ID=

WECHAT_SECRET=

# ==================== 钉钉 ====================
DINGTALK_APP_KEY=
DINGTALK_APP_SECRET=
DINGTALK_ACCESS_TOKEN=

# ==================== 飞书 ====================
FEISHU_APP_ID=
FEISHU_APP_SECRET=
FEISHU_ACCESS_TOKEN=

# ==================== 阿里云 ====================
ALIYUN_ACCESS_KEY_ID=
ALIYUN_ACCESS_KEY_SECRET=

# ==================== Google Calendar ====================
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REFRESH_TOKEN=
```

**安全要求**：
1. 此文件必须加入 `.gitignore`
2. 敏感信息使用 AES-256 加密存储
3. 生产环境使用环境变量注入，禁止硬编码
