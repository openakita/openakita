# 工作流模板：数据报表自动生成

**模板 ID**: WF-04  
**分类**: 数据分析  
**版本**: 1.0  
**创建时间**: 2026-03-11 07:07  
**状态**: 已启用

---

## 一、模板概述

定时从多数据源提取数据，生成可视化报表并发送

### 适用场景
- 数据分析行业
- 需要自动化处理重复性工作流程
- 多系统协同作业场景

---

## 二、触发条件

支持以下触发方式：

- ✅ 定时任务 (每日 9:00)
- ✅ 手动触发

---

## 三、执行步骤

| 步骤 | 动作 | 系统/服务 | 预计耗时 |
|------|------|-----------|----------|
| 1 | 提取业务数据 | 数据库/API | 自动 |
| 2 | 数据清洗 | ETL | 自动 |
| 3 | 计算指标 | 数据分析 | 自动 |
| 4 | 生成图表 | 可视化引擎 | 自动 |
| 5 | 发送报表 | 邮件/IM | 自动 |

---

## 四、系统集成

需要以下系统/平台集成：

- 🔌 MySQL

- 🔌 Excel
- 🔌 钉钉
- 🔌 企业邮箱

## 五、配置参数

### 5.1 必填参数
| 参数名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| workflow_name | string | 工作流名称 | 数据报表自动生成 |
| trigger_type | string | 触发类型 | event/timer |
| timeout | integer | 超时时间 (秒) | 300 |

### 5.2 可选参数
| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| retry_count | integer | 3 | 失败重试次数 |
| notify_on_complete | boolean | true | 完成时通知 |
| log_level | string | info | 日志级别 |

---

## 六、异常处理

### 6.1 常见异常
| 异常类型 | 处理策略 | 通知方式 |
|----------|----------|----------|
| 系统连接失败 | 自动重试 3 次 | 邮件 + 短信 |
| 数据格式错误 | 跳过并记录 | 邮件 |
| 超时 | 终止流程 | 邮件 + 短信 |
| 审批拒绝 | 终止流程 | 邮件 |

### 6.2 告警规则
- 连续失败 3 次 → 电话通知管理员
- 处理时长超过 10 分钟 → 邮件告警
- 系统异常 → 短信 + 邮件

---

## 七、监控指标

| 指标 | 计算方式 | 告警阈值 |
|------|----------|----------|
| 成功率 | 成功次数/总次数 | < 95% |
| 平均处理时长 | 总时长/成功次数 | > 5 分钟 |
| 失败率 | 失败次数/总次数 | > 5% |
| 系统可用性 | 正常运行时间/总时间 | < 99% |

## 八、使用示例

### 8.1 API 调用示例

```json
{
  "workflow_id": "WF-04",
  "action": "execute",
  "parameters": {
    "workflow_name": "数据报表自动生成",
    "trigger_type": "event",
    "timeout": 300
  }
}
```

### 8.2 可视化配置

1. 打开工作流编排器
2. 选择模板「数据报表自动生成」
3. 配置触发条件和集成参数
4. 保存并启用
