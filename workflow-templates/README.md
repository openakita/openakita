## 模板结构说明

每个模板文件包含以下标准结构：

```markdown
# 模板名称

## 适用场景
- 目标用户
- 使用频率
- 业务价值

## 触发条件
- 触发事件
- 输入数据

## 执行流程
1. 步骤一
2. 步骤二
3. 步骤三

## 集成 API
- 钉钉/企微/飞书
- 第三方服务

## 输出结果
- 最终产出
- 通知对象
```
