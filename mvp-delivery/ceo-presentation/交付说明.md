## 🎯 MVP 核心信息摘要

### 产品定位
> **执行型 Agent** —— 让 AI 真正帮你干活，不只是聊天

### 目标用户
- 企业规模：50-500 人
- 核心痛点：重复性工作多、人力成本高、传统 RPA 太贵
- 典型场景：订单处理、报表生成、审批流程

### 核心优势
1. **私有化部署** - 数据不出内网，符合企业安全要求
2. **工作流模板库** - 50+ 预置模板，3 分钟上手
3. **性价比** - ¥9,800/年起，1/10 传统 RPA 成本
4. **快速部署** - 3 分钟完成配置，15 分钟学会使用

## 📊 技术栈说明

### 核心技术
- **后端**: Python 3.11+ (FastAPI, asyncio)
- **前端**: React 18 + TypeScript + Vite 6
- **向量数据库**: Qdrant (性能高、部署简单)
- **异步队列**: Celery + Redis
- **数据库**: PostgreSQL
- **部署**: Docker + Kubernetes (可选)

### 差异化技术
- **三层记忆系统**: 核心档案 + 语义记忆 + 原始对话
- **Ralph 循环**: 永不放弃的任务执行机制
- **技能系统**: 可扩展的能力模块
- **MCP 集成**: 支持外部工具协议
