## 🔐 凭据记录表

> ⚠️ **安全提醒**: 真实凭据请勿直接填写在此文档！  
> 此文档仅记录**非敏感信息**，敏感 Key/Secret 通过加密渠道发送给技术负责人。

### 1. 企业微信

| 字段 | 值 | 敏感性 | 记录方式 |
|------|-----|--------|----------|
| CorpID | - | 中 | 填写在此 |
| AgentID | - | 中 | 填写在此 |
| Agent Secret | - | 🔴 高 | 加密渠道发送 |
| 应用名称 | - | 低 | 填写在此 |
| 权限范围 | message.send, agent.get | 低 | 填写在此 |

**申请步骤**:
1. 登录企业微信管理后台 (https://work.weixin.qq.com)
2. 应用管理 → 自建 → 创建应用
3. 记录 CorpID、AgentID、Secret
4. 配置应用权限：发送消息

### 3. 阿里云 (邮件/短信/OSS)

| 字段 | 值 | 敏感性 | 记录方式 |
|------|-----|--------|----------|
| AccessKey ID | - | 🔴 高 | 加密渠道发送 |
| AccessKey Secret | - | 🔴 高 | 加密渠道发送 |
| RAM 子账号名称 | - | 低 | 填写在此 |
| 邮件推送 Account | - | 中 | 填写在此 |
| 短信签名 | - | 低 | 填写在此 |
| OSS Bucket 名称 | - | 低 | 填写在此 |

**申请步骤**:
1. 登录阿里云控制台 (https://aliyun.com)
2. RAM 访问控制 → 创建用户 (勾选"为新生成用户创建 AccessKey")
3. 权限策略：AliyunDYSMSFullAccess, AliyunOSSFullAccess, AliyunDirectMailFullAccess
4. 分别开通邮件推送、短信服务、OSS 服务
5. 创建 OSS Bucket (选择区域：华东 1-杭州)
