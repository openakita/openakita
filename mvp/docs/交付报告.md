## 三、技术实现说明

### 3.1 工作流编排器前端

**核心组件**:
```typescript
WorkflowCanvas
├── NodeLibrary (节点库)
│   ├── HTTP 节点
│   ├── Condition 节点
│   ├── Loop 节点
│   ├── Transform 节点
│   └── Notification 节点
├── Canvas (画布)
│   ├── ReactFlow
│   ├── Controls
│   └── Background
└── PropertyPanel (属性面板)
    ├── 节点名称编辑
    └── 节点描述编辑
```

**性能优化**:
- 使用 React Flow 虚拟化渲染
- 节点状态使用 Zustand 管理
- 连线使用 smoothstep 算法

### 3.2 工作流引擎后端

**核心类**:
```python
WorkflowEngine
├── _load_templates()      # 加载模板
├── _parse_template()      # 解析模板
├── list_templates()       # 列出模板
├── get_template()         # 获取模板详情
├── execute()              # 执行工作流
└── _run_instance()        # 运行实例
```

**模板解析流程**:
```
Markdown 文件 → 提取元数据 → 解析执行步骤 → 提取触发条件 → 提取集成系统 → Dict 对象
```

### 3.3 API 集成层

**架构设计**:
```python
WorkflowAPI (高级封装)
    ↓
APIAdapter (统一适配器)
    ↓
┌─────────────────┐
│ Environment.MOCK │ → _mock_response()
│ Environment.REAL │ → _real_request()
└─────────────────┘
```

**环境切换**:
```python
# Mock 环境（开发测试）
client = WorkflowAPI(environment=Environment.MOCK)

# 真实环境（生产）
client = WorkflowAPI(environment=Environment.REAL)
```
