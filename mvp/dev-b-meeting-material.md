### 1.3 技术实现方案

**工作流编排器**:
- 前端：React Flow + TypeScript + Tailwind CSS
- 状态管理：Zustand（轻量级，适合复杂交互）
- 画布性能：虚拟滚动 + 懒加载，支持 100+ 节点流畅渲染

**API 集成层**:
- 统一适配器模式，标准化接口
- OAuth 2.0 认证管理
- 速率限制 + 重试机制

**邮件处理**:
- 使用 `imaplib` + `smtplib`（Python 标准库）
- 支持 Gmail/Outlook/腾讯企业邮/阿里企业邮
- HTML 邮件解析 + 模板引擎

**报表生成**:
- 数据源：PostgreSQL + Redis 缓存
- 导出：`openpyxl` (Excel) + `reportlab` (PDF)
- 定时任务：Celery Beat
