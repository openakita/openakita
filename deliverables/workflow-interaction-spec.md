## 三、节点类型与交互

### 3.1 触发器节点（绿色边框）

**用途**: 工作流启动入口

| 类型 | 图标 | 配置项 | 示例 |
|------|------|--------|------|
| 定时触发 | 🕐 | Cron 表达式 | 每天 09:00 |
| 事件触发 | ⚡ | 事件类型 | 新订单产生 |
| Webhook | 🔗 | API 端点 | POST /webhook |
| 手动触发 | 👆 | 无 | 点击执行 |

**交互流程**:
1. 从左侧面板拖入画布
2. 点击节点，右侧展开配置面板
3. 选择触发类型，填写参数
4. 配置完成后显示绿色对勾

### 3.2 动作节点（蓝色边框）

**用途**: 执行具体任务

| 类型 | 图标 | 配置项 | 示例 |
|------|------|--------|------|
| 发送邮件 | 📧 | 收件人/主题/内容 | 订单确认邮件 |
| 报表生成 | 📊 | 数据源/图表类型 | 销售日报 |
| 数据整理 | 🗂️ | 输入/输出字段 | 数据清洗 |
| API 调用 | 🌐 | URL/方法/参数 | 调用钉钉 API |

### 3.3 条件分支节点（黄色菱形）

**用途**: 逻辑判断，分流处理

**视觉特征**:
- 菱形形状，黄色边框 (#F59E0B)
- 1 个输入端口（左侧）
- 2 个输出端口（右侧，标记 True/False)

**配置项**:
- 条件表达式：`金额 > 1000`
- 支持字段引用：`{{order.amount}}`
- 操作符：`=`, `!=`, `>`, `<`, `>=`, `<=`, `contains`

### 3.4 循环节点（紫色边框）

**用途**: 批量处理，遍历列表

**配置项**:
- 遍历对象：数组/列表
- 循环变量名：`item`, `index`
- 终止条件（可选）

### 3.5 数据转换节点（橙色边框）

**用途**: 字段映射，数据格式转换

**配置项**:
- 字段映射表：`order_id → 订单编号`
- 转换函数：`toUpperCase()`, `formatDate()`
- 自定义表达式（JavaScript）

## 四、画布交互规范

### 4.1 节点操作

| 操作 | 交互方式 | 反馈 |
|------|----------|------|
| 拖入节点 | 从左侧面板拖拽到画布 | 节点跟随鼠标，半透明预览 |
| 移动节点 | 拖拽节点 | 节点实时跟随，连线同步更新 |
| 连线 | 从输出端口拖到输入端口 | 贝塞尔曲线实时绘制 |
| 删除节点 | 选中后按 Delete | 节点缩小淡出动画 (200ms) |
| 复制节点 | Ctrl+C / Ctrl+V | 副本出现在原节点右下方 |
| 多选 | Shift+ 点击或框选 | 多个节点同时高亮 |

### 4.2 画布导航

| 操作 | 交互方式 |
|------|----------|
| 平移 | 鼠标右键按住拖动 或 空格 + 左键拖动 |
| 缩放 | 鼠标滚轮（范围 25%-200%） |
| 适应屏幕 | 双击空白处 或 Ctrl+0 |
| 重置视图 | 点击工具栏"适应屏幕"按钮 |

### 4.3 连接线样式

| 状态 | 样式 | 颜色 | 线宽 |
|------|------|------|------|
| 默认 | 实线 | #9CA3AF | 2px |
| 选中 | 实线 | #2563EB | 3px |
| 数据流动 | 虚线动画 | #2563EB | 2px |
| 错误 | 虚线 | #EF4444 | 2px |

## 五、配置面板交互

### 5.1 面板结构

```
┌─────────────────────────────┐
│ 节点名称               ✕    │
├─────────────────────────────┤
│ 基础配置                    │
│ [输入框] 节点名称           │
│ [下拉框] 类型选择           │
│                             │
│ 高级配置                    │
│ [开关] 启用重试             │
│ [输入框] 重试次数           │
│                             │
│ ─────────────────────────── │
│      [取消]    [保存]       │
└─────────────────────────────┘
```

### 5.2 配置项类型

| 类型 | 组件 | 验证规则 |
|------|------|----------|
| 文本输入 | Input | 必填，最大 50 字符 |
| 下拉选择 | Select | 必填 |
| 代码编辑 | CodeEditor | 语法检查 |
| 字段映射 | FieldMapper | 键值对验证 |
| 开关 | Switch | 布尔值 |
| 数字输入 | NumberInput | 范围验证 |

### 5.3 自动保存机制

- 配置变更 → 实时预览
- 点击画布空白处 → 自动保存
- 保存成功 → 节点显示绿色对勾
- 保存失败 → 显示错误提示

## 六、核心用户流程

### 6.1 创建新工作流

```
1. 点击"新建工作流"按钮
   ↓
2. 输入名称和描述（弹窗）
   ↓
3. 进入空白画布
   ↓
4. 拖入触发器节点
   ↓
5. 配置触发器
   ↓
6. 拖入动作节点并连线
   ↓
7. 配置动作节点
   ↓
8. 点击"保存"
```

**预计耗时**: 3 分钟内（新手用户）

### 6.2 配置节点

```
1. 点击画布中的节点
   ↓
2. 右侧自动展开配置面板
   ↓
3. 修改配置项（实时预览）
   ↓
4. 点击画布空白处（自动保存）
   ↓
5. 节点显示绿色对勾
```

### 6.3 测试工作流

```
1. 点击"测试"按钮
   ↓
2. 选择测试数据
   ↓
3. 开始测试
   ↓
4. 逐节点高亮执行
   ↓
5. 显示结果报告
```

## 七、错误处理

### 7.1 配置错误

| 场景 | 表现 | 解决方案 |
|------|------|----------|
| 必填项为空 | 节点边框红色警告 | 填写完整配置 |
| 格式错误 | 输入框下方红色提示 | 修正格式 |
| 表达式语法错误 | CodeEditor 标红 | 修复语法 |

### 7.2 连接错误

| 场景 | 表现 | 解决方案 |
|------|------|----------|
| 类型不匹配 | 端口不高亮，无法连接 | 检查节点类型 |
| 循环依赖 | 连线显示红色虚线 | 断开循环连接 |
| 孤立节点 | 节点显示黄色警告 | 连接或移除 |

### 7.3 执行错误

| 场景 | 表现 | 解决方案 |
|------|------|----------|
| API 调用失败 | 节点红色背景 | 检查 API 配置 |
| 数据格式错误 | 显示错误详情 | 修正数据 |
| 超时 | 显示超时提示 | 增加超时时间 |

### 8.1 断点定义

| 断点 | 宽度 | 布局调整 |

| Desktop | ≥1440px | 三栏布局（完整） |
