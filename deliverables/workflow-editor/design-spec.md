## 五、核心组件设计

### 5.1 节点卡片 (Node Card)

**尺寸**: 240px × 自动高度  
**结构**:
- 节点头部 (图标 + 标题)
- 节点内容区 (描述/配置预览)
- 输入/输出端口 (左右两侧)

**状态**:
- 默认：白色背景，灰色边框
- 悬停：阴影加深，轻微上浮
- 选中：蓝色边框，外发光效果
- 错误：红色边框，闪烁动画

### 5.2 连接线 (Connection Line)

**样式**:
- 默认：灰色实线 (#9CA3AF), 2px
- 选中：蓝色实线 (#2563EB), 3px
- 数据流动：蓝色虚线动画
- 错误：红色虚线 (#EF4444), 2px

**交互**:
- 从输出端口拖拽到输入端口
- 贝塞尔曲线自动美化
- 支持点击删除

### 5.3 配置面板 (Config Panel)

**宽度**: 320px  
**结构**:
- 面板标题 (节点图标 + 名称)
- 基础配置区 (名称/描述/类型)
- 高级配置区 (重试/超时/错误处理)
- 操作按钮区 (保存/取消/测试)

## 六、交互说明

### 6.1 节点操作

| 操作 | 交互方式 | 说明 |
|------|----------|------|
| 创建节点 | 从左侧拖入画布 | 显示预览阴影 |
| 移动节点 | 拖拽节点头部 | 实时跟随鼠标 |
| 连线 | 从输出端口拖到输入端口 | 自动吸附 |
| 删除节点 | 选中后按 Delete | 二次确认 |
| 复制节点 | Ctrl+C / Ctrl+V | 快速复制 |
| 多选 | Shift+ 点击或框选 | 批量操作 |

### 6.2 画布导航

| 操作 | 交互方式 |
|------|----------|
| 平移 | 鼠标右键按住拖动 或 空格 + 左键拖动 |
| 缩放 | 鼠标滚轮 或 Ctrl+ 滚轮 |
| 适应屏幕 | 双击空白处 或 Ctrl+0 |
| 重置视图 | 点击工具栏"适应屏幕"按钮 |

### 6.3 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存工作流 |
| Ctrl+R | 运行工作流 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Delete | 删除选中节点 |
| Ctrl+C/V | 复制/粘贴 |
| 方向键 | 移动选中节点 (10px 步进) |
| Shift+ 方向键 | 快速移动 (50px 步进) |

## 七、用户引导流程 (5 步上手)

### 7.1 流程概览

| 步骤 | 内容 | 预计时间 | 跳过机制 |
|------|------|----------|----------|
| Step 1 | 欢迎页 - 价值主张 + 案例展示 | 30 秒 | 无 |
| Step 2 | 选择行业 - 4 个行业卡片 | 30 秒 | "稍后选择" |
| Step 3 | 模板预览 - 流程图可视化 | 60 秒 | 无 |
| Step 4 | 关键配置 - 表单式配置 | 60 秒 | "稍后设置" |
| Step 5 | 测试运行 - 执行动画 + 成功反馈 | 30 秒 | 无 |

### 7.2 设计细节

- **进度条**: 顶部显示 5 步进度，已完成步骤显示绿色
- **帮助提示**: 每步右下角"？"按钮，点击查看详细说明
- **返回上一步**: 支持返回修改选择
- **跳过补充**: 跳过的配置可在设置中补充

## 八、数据看板界面

### 8.1 核心指标卡片

| 指标 | 展示内容 | 更新频率 |
|------|----------|----------|
| 今日执行次数 | 数字 + 趋势百分比 | 实时 |
| 成功率 | 百分比 + 环比 | 实时 |
| 节省时间 | 小时数 + 累计值 | 每小时 |
| 活跃工作流 | 数字 + 列表 | 实时 |

### 8.2 图表组件

1. **执行趋势图** (折线图)
   - 支持 7 天/30 天切换
   - 实时数据刷新 (<3 秒)

2. **工作流排行** (柱状图)
   - Top 10 工作流
   - 显示执行次数和状态

3. **错误分布** (饼图)
   - 按错误类型分类
   - 支持点击筛选

### 8.3 交互功能

- 时间范围选择 (今日/本周/本月)
- 数据导出 (Excel/PDF)
- 自定义看板 (拖拽组件)
- 实时刷新 (可配置频率)

## 十、技术实现建议

### 10.1 推荐技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | React 18 + TypeScript | 组件化开发，类型安全 |
| 流程图库 | React Flow | 专业工作流编辑器组件 |
| 样式方案 | Tailwind CSS | 快速开发，易维护 |
| 状态管理 | Zustand | 轻量级状态管理 |
| 图表库 | Recharts | React 友好，易定制 |

### 10.2 性能优化

- **虚拟滚动**: 仅渲染可视区域节点
- **防抖处理**: 缩放/平移 100ms 防抖
- **增量保存**: 工作流数据增量保存
- **本地缓存**: 历史记录本地缓存 (最近 50 步)

### 10.3 可访问性

- 所有图标按钮带 `aria-label`
- 键盘导航完整支持
- 文字对比度 ≥ 4.5:1 (AA 标准)
- 焦点状态明显可见 (3px 蓝色外框)
