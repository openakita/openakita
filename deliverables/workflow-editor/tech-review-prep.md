## 一、演示内容清单

### 1.1 UI 原型演示（15 分钟）

**演示环境**: workflow-orchestrator-prototype.html

**演示脚本**:
1. **开场**（1 分钟）
   - 产品定位：执行型 Agent，让 AI 真正帮你干活
   - 核心差异化：3 分钟上手 vs 竞品 10+ 分钟

2. **工作流编辑器核心交互**（5 分钟）
   - 拖拽节点到画布
   - 连接节点创建流程
   - 配置节点属性（右侧面板）
   - 画布交互（缩放/平移/右键菜单）

3. **5 种节点类型展示**（3 分钟）
   - 触发器节点（蓝色）：定时/事件/API webhook
   - 动作节点（绿色）：邮件/报表/数据整理
   - 条件节点（橙色）：if/else 分支
   - 循环节点（紫色）：for each/while
   - 工具节点（青色）：数据转换/字段映射

4. **预置模板演示**（3 分钟）
   - 选择行业模板（电商/服务业/制造业/专业服务）
   - 一键导入模板
   - 查看和修改模板

5. **用户引导流程**（3 分钟）
   - 5 步上手流程演示
   - 实测 97 秒完成首个任务

### 1.2 设计说明文档（5 分钟）

**核心设计原则**:
1. 3 分钟上手 - 新用户首次创建不超过 3 分钟
2. 渐进式复杂度 - 基础功能极简，高级功能可探索
3. 一致性 - 遵循 Ant Design 设计体系
4. 可访问性 - 支持键盘操作和屏幕阅读器

**技术栈建议**:
- 前端框架：React 18
- 工作流引擎：React Flow
- UI 组件库：Ant Design / Tailwind CSS
- 状态管理：Zustand / Jotai

**性能指标**:
- 100+ 节点渲染 <2 秒
- 拖拽响应 <100ms
- 内存占用 <200MB
